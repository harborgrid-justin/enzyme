## Core Concepts

### 1. Feature-Based Architecture

Harbor React uses vertical slice architecture where each feature is self-contained:

```

â””â”€â”€ [feature]/
    â”œâ”€â”€ components/    # UI components
    â”œâ”€â”€ hooks/         # Data fetching hooks
    â”œâ”€â”€ wiring/        # API clients and view models
    â”œâ”€â”€ config.ts      # Feature configuration
    â”œâ”€â”€ model.ts       # TypeScript types
    â””â”€â”€ index.ts       # Public API
```

See: [Features Guide](./FEATURES.md)

### 2. Enterprise Configuration Hub

Single source of truth for all configuration:

```typescript
import {
  env, ROUTES, API_CONFIG, STORAGE_KEYS,
  COLORS, TIMING, initializeConfig
} from '@/config';
```

See: [Configuration Guide](./CONFIGURATION.md)

## Technology Stack

| Category | Technology | Purpose |
|----------|------------|---------|
