# Generate Code with Enzyme

Enzyme provides powerful code generators to accelerate your development.

## Available Generators

### Components
Generate React components with proper TypeScript types:
- Functional components
- Page components
- Layout components
- With or without props

**Shortcut:** `Ctrl+Alt+E C` (Windows/Linux) or `Cmd+Alt+E C` (Mac)

### Features
Create complete feature modules with:
- Components
- Routes
- State management
- API clients
- Tests

**Shortcut:** `Ctrl+Alt+E F`

### Routes
Generate route definitions:
- Lazy-loaded components
- Route guards
- Loaders and actions
- Nested routes

**Shortcut:** `Ctrl+Alt+E R`

### Hooks
Create custom React hooks:
- State management hooks
- Effect hooks
- Reusable logic

**Shortcut:** `Ctrl+Alt+E H`

### Stores
Generate Zustand stores:
- TypeScript types
- DevTools integration
- Persistence
- Middleware

**Shortcut:** `Ctrl+Alt+E S`

### Tests
Generate test files:
- Component tests
- Hook tests
- Integration tests
- Using Vitest or Jest

**Shortcut:** `Ctrl+Alt+E T`

## Generator Wizard

For a guided experience, open the Generator Wizard:

[Open Generator Wizard](command:enzyme.panel.showGeneratorWizard)

This provides a step-by-step interface to create any code structure.
