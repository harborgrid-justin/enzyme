# Explore Your Enzyme Project

The Enzyme explorer helps you navigate and understand your project structure.

## Explorer Views

### Features View
See all your feature modules:
- Feature metadata
- Access control settings
- Component tree
- Feature flags

**Actions:**
- Click to navigate to feature
- Right-click for options
- Generate new features

### Routes View
Visualize your application routes:
- Route hierarchy
- Path parameters
- Route guards
- Lazy-loaded components

**Features:**
- Click to open route file
- See route conflicts
- Validate route configuration

### Components View
Browse all components:
- Organized by feature
- Component dependencies
- Usage statistics
- Quick navigation

### Stores View
Manage state stores:
- Store structure
- State shape
- Actions and selectors
- DevTools integration

### API Clients View
Track API endpoints:
- HTTP methods
- Request/response types
- Error handling
- Usage examples

### Performance View
Monitor application health:
- Bundle size
- Component render counts
- Memory usage
- Load times

## Navigation Tips

- Use the search bar to find specific items
- Right-click for context actions
- Use keyboard shortcuts for quick access
- Refresh views to update data

[Refresh Explorer](command:enzyme.explorer.refresh)
